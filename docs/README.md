# Quick Intro

Spotify-api.js is a quick wrapper to interact with spotify api!

# Installation

```bash
npm i spotify-api.js
```

# How to Start

- TypeScript
```ts
import { Client } from 'spotify-api.js'
```

- JavaScript
```js
const { Client } = require('spotify-api.js')
```
